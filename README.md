# Task Manager Spring Boot Course

**Gerenciador de Tarefas** — API REST desenvolvida com **Java** e **Spring Boot** como projeto de curso e portfólio.

## Status
**Status:** Concluído / Pronto para demonstração

## Tecnologias
- **Java** (ver versão no `pom.xml`)
- **Spring Boot**
- **Spring Data JPA** e **Hibernate**
- **H2** para desenvolvimento; **PostgreSQL** como opção de produção
- **Maven** como ferramenta de build
- **Lombok** para redução de boilerplate

## Pré Requisitos
- **JDK 8+** (confira a versão exata no `pom.xml`)
- **Maven 3.6+**
- (Opcional) **Docker** para containerização
